# HR-Employee-Attrition-and-Performance-data
## **Project Overview**
This project analyzes employee performance, satisfaction and attrition patterns using an HR dataset. The goal is to identify key factors influencing employee turnover and performance to support data driven HR decisions.
## **Tools used**
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Jupyter Notebook
## **Dataset**
- Domain- Human Resource Analytics
- Source- UCI Machine Learning Repository
- Size- 311 rows and 35 columns
- Key Features- Department, Performance Score, Employee Satisfaction, Salary, Absences, Employment Status, Hire &Termination dates.
## **Methodology**
- Data loading and inspection
- Data cleaning and preprocessing
  - Handling missing values
  - Data type conversion
  - Derived columns(tenure, absence level, employment status)
- Exploratory Data Analysis (EDA)
  - Univariate, bivariate and multivariate analysis
  - Groupby, pivot tables and correlation analysis
- Data visualizations for insights
## **Key Insights**
- Higher performance scores are associated with higher employee satisfaction
- Production department shows the highest employee attrition
- Salary has very weak correlation with employee absences
- Voluntary attrition is more common than termination for cause
## **Visualizations included**
- Performance score distribution
- Department wise attrition analysis
- Salary vs absences correlation
- Satisfaction trends across performance levels
## **Conclusion**
The analysis highlights meaningful relationships between performance, satisfaction and attrition. These insights can help HR teams improve employee retention and workforce planning.
